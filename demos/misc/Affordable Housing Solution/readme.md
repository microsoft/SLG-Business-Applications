Overview

This repository provides a complete solution for building and exploring a sample application using Microsoft Power Platform. 

It includes:

Solution File: Contains the sample app and schema for quick deployment.
Schema Definition File: Defines the data structure for the application.
Lab Guides: Step-by-step instructions for hands-on learning.
Sample Data Spreadsheets: Pre-populated data for testing and demonstrations.
